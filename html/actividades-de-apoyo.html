Actividades de apoyo

¿Para qué?

Antes de realizar los proyectos, realizarás una serie de actividades que te ayudarán a comprender mejor los conceptos básicos.

Estas actividades incluyen:

  • Diccionario de términos: para conocer y entender el significado de cada término.
  • Vídeo explicativo sobre los sistemas operativos con una actividad relacionada
  • Juego de adivinar la palabra oculta para familiarizarte con los términos

Diccionario de términos

Sistema operativos

Software que permite que un dispositivo funcione, gestionando el hardware y permitiendo ejecutar programas y aplicaciones.

Red informática

Conjunto de dispositivos conectados entre sí que pueden compartir información y recursos.

Internet

Red global que conecta millones de dispositivos en todo el mundo y permite el acceso a información y servicios digitales.

Dirección IP

Número único que identifica a un dispositivo dentro de una red para poder comunicarse con otros.

Router

Dispositivo que conecta una red local con Internet y permite que varios dispositivos accedan a la red.

 Protocolo de red

Conjunto de normas que permiten la comunicación entre dispositivos en una red (por ejemplo, HTTP o TCP/IP).

LAN (Red de área local)

Red que conecta dispositivos en un espacio reducido, como una casa, un aula o una oficina.

WAN (Red de área amplia)

Red que conecta dispositivos a gran distancia, como Internet.

Malware

Software malicioso diseñado para dañar un dispositivo o robar información.

Phishing

Intento de engaño para obtener información personal (contraseñas, datos bancarios) haciéndose pasar por una entidad fiable.

Cortafuegos (Firewall)

Sistema de seguridad que controla el tráfico de red y protege los dispositivos de accesos no autorizados.

Antivirus

Programa que detecta, bloquea y elimina software malicioso.

Cifrado

Proceso que protege la información transformándola en un formato que solo puede ser leído por quien tiene la clave.
